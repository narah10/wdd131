Project Site Plan

Site Name: MovieVault

This name represents a website that is focused on displaying different data for movies.

Site Purpose for MovieVault

Movie enthusiasts can explore and search a vast collection of films from around the world. There will be a page dedicated to top rated movies for easy navigation for users wanting to go straight into seeing what is popular out there.

Scenarios

  • What are some highly recommended movies?
  • Where can I find family-friendly movies that are suitable for all ages?
  • Where can I search for a certain movie and get information?
